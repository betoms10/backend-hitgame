## Description

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy to server using PM2

```bash

$ npm install

$ npm run build

# copy  all folders, except src/ (typescript files) to destination folder on server.

# update file .env with production environment variables

# run start.bat script to start application



```
